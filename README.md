## 项目名称
  demo

## 项目使用的技术栈
  > React + TS + React-Router + redux + react-redux + fetch&axios实现移动端页面
  > 项目使用TS实现,需要TS基础

## 项目文件分工（个人习惯）
  ```
    > components   ---> 主要目录组件 例如：底部导航tab都放在components
    > pages        ---> 主要的页面部分，例如，底部导航的一级页面
    > plugins      ---> 复用的组件
    > views        ---> 所有的二级页面三级页面的存放目录
  ```
